# DataHack 2026 Evaluation Checklist

Use this checklist to ensure our project meets all evaluation criteria.

## 1. Innovation & Originality (20%)

- [ ] Novel problem definition or perspective
- [ ] Inventive analytical strategies
- [ ] Creative data utilization revealing unexpected insights
- [ ] Unique approaches to standard problems

**Notes:**


## 2. Impact & Practical Feasibility (30%)

- [ ] Solution has genuine real-world applicability
- [ ] Addresses core problem effectively
- [ ] Creates measurable value
- [ ] Considers economic constraints
- [ ] Considers social constraints
- [ ] Scalable to other contexts
- [ ] Aligns with ESG principles (Environmental, Social, Governance)

**Notes:**


## 3. Analytical Rigor & Data Competency (30%)

- [ ] Effective data integration from multiple sources
- [ ] Methodologically appropriate analytical models
- [ ] Evidence-based justification of recommendations
- [ ] Awareness of data quality issues documented
- [ ] Limitations clearly stated
- [ ] Sound statistical/technical execution

**Notes:**


## 4. Presentation, Visualization & Interdisciplinary Collaboration (20%)

- [ ] Clear and effective visualizations
- [ ] Strong data storytelling
- [ ] Persuasive communication
- [ ] Meaningful contributions from STEM members documented
- [ ] Meaningful contributions from non-STEM members documented
- [ ] All data sources explicitly cited
- [ ] Detailed attribution of individual team contributions
- [ ] Professional presentation quality

**Notes:**


## Submission Requirements

- [ ] Completed hackathon event
- [ ] Participated in mandatory final presentation
- [ ] All data sources disclosed
- [ ] Each member's contributions specified
